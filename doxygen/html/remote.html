Current setting getter

:ISET?: Returns current setting .

Current setting setter

:ISET:value Set current value (in mA) .
See Return value.

Calibration

:CAL:toggle Turns ON or OFF the logging feature.
:CAL:section:slope,offset section could be V, C or D, standing for Voltage, Current, DAC. slope and offset are floating point values, with US decimal '.' separator. These values could be calculated using the LibreOffice's spreadsheet file: aDCLoadCalibration.ods.
:CAL:SAVE Backup calibation datas into EEPROM. .
See Return value .
See Calibration Process

DAC value setter (calibration purpose)

:DAC:value Set DAC value (from 0 to 4095). This command has no effect outside calibration mode .
See Calibration .
See Calibration Process

Logging enability

:LOG:toggle Turns ON or OFF the logging feature.
If toggle value is not specified, a single logging line is returned. .
See Return value.

Return value

:value:status:

  • Where:
    • value if any expected. INVALID on unknown command.
    • status could be OK on success or ERR on failure.
